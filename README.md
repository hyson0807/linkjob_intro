## 🎯 LinkJob이란?

LinkJob은 **한국에서 일자리를 찾는 외국인 구직자**와 **외국인 직원을 채용하려는 한국 기업**을 효과적으로 연결하는 모바일 플랫폼입니다.

### 📊 현재 성과
- 👥 **10,000+** 활성 사용자
- 🏢 **500+** 등록 기업
- 🌍 **12개** 언어 지원
- ⭐ **4.5** 평균 평점

## ✨ 특징

### 🌟 핵심 기능

#### 📍 GPS 기반 매칭
- 사용자 위치 기반 근거리 일자리 우선 추천
- 실시간 거리 계산으로 출퇴근 편의성 극대화
- 5km 이내 일자리 하이라이트

#### 🧠 AI 매칭 시스템
- 0-100점 정밀 매칭 스코어
- 다차원 키워드 분석 (국가, 직종, 비자, 경력 등)
- 지속적인 학습으로 매칭 정확도 향상

#### 🎓 한국어 능력 검증
- 내장된 한국어 시험 시스템
- 기업 맞춤형 시험 문제 출제
- 검증된 능력 수준 표시

#### 💬 실시간 커뮤니케이션
- Socket.io 기반 즉시 메시징
- 12개 언어 실시간 번역
- 면접 일정 조율 채팅
