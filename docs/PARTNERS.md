# 파트너십 안내 🤝

## LinkJob 파트너 프로그램

LinkJob과 함께 외국인 채용 시장을 혁신하고 성장하세요.

## 파트너십 유형

### 🏢 기업 파트너
**대상**: 정기적으로 외국인 채용이 필요한 기업

**혜택**:
- 전담 어카운트 매니저 배정
- 맞춤형 채용 솔루션 제공
- 우선 매칭 서비스
- 대량 채용 할인
- 채용 컨설팅 지원

**조건**:
- 월 5명 이상 채용 예정
- 6개월 이상 파트너십 약정

### 🎓 교육 기관 파트너
**대상**: 대학, 한국어학원, 직업훈련기관

**혜택**:
- 학생 취업 지원 프로그램
- 캠퍼스 리크루팅 지원
- 학생 전용 프리미엄 혜택
- 취업률 통계 제공

**조건**:
- 정식 인가 교육기관
- 외국인 학생 50명 이상

### 🌏 정부/공공기관 파트너
**대상**: 고용노동부, 지자체, 공공기관

**프로그램**:
- 외국인 고용 지원 사업 협력
- 정착 지원 프로그램 운영
- 정책 데이터 제공
- 공동 이벤트 개최

### 💼 인력 에이전시 파트너
**대상**: 헤드헌팅, 인력파견 업체

**혜택**:
- API 연동 지원
- 화이트라벨 서비스
- 수수료 쉐어링
- 공동 마케팅

**조건**:
- 정식 등록 업체
- 월 10건 이상 알선 실적

## 파트너 혜택

### 🎯 비즈니스 혜택
- **우선 노출**: 파트너 뱃지 및 상위 노출

### 📢 마케팅 혜택

- **사례 공유**: 성공 스토리 홍보
- **이벤트 초대**: 네트워킹 행사
- **PR 지원**: 보도자료 배포

### 🛠 기술 지원
- **전담 지원팀**: 24/7 기술 지원
- **맞춤 개발**: 커스터마이징 지원
- **교육 프로그램**: 직원 교육 제공
- **우선 업데이트**: 신기능 우선 제공

## 성공 사례

### 제조업 A사
- **도전**: 생산직 외국인 대량 채용 필요
- **솔루션**: 맞춤형 채용 페이지 + 전담 매니저
- **결과**: 3개월 만에 50명 채용 성공

### B 대학교
- **도전**: 졸업생 취업률 향상 필요
- **솔루션**: 캠퍼스 전용 채널 개설
- **결과**: 외국인 학생 취업률 30% 상승

### C 지자체
- **도전**: 지역 중소기업 인력난 해결
- **솔루션**: 지역 특화 매칭 프로그램
- **결과**: 200개 기업 인력 문제 해결
