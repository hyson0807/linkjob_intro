# 자주 묻는 질문 (FAQ)

## 📱 일반

### LinkJob은 무엇인가요?
LinkJob은 한국에서 일자리를 찾는 외국인과 외국인 직원을 채용하려는 기업을 연결하는 모바일 플랫폼입니다.

### 서비스 이용료가 있나요?
- **구직자**: 무료
- **기업**: 기본 무료, 프리미엄 기능 유료

### 어떤 기기에서 사용할 수 있나요?
- iOS 13.0 이상
- Android 6.0 이상
- 웹 브라우저 (제한적)

## 👤 구직자

### 한국어를 못해도 사용할 수 있나요?
네! 12개 언어를 지원하며, 실시간 번역 기능도 제공합니다.

### 이력서는 어떻게 등록하나요?
프로필 작성이 곧 이력서입니다. 프로필을 완성하면 자동으로 이력서가 생성됩니다.

### 여러 회사에 동시 지원이 가능한가요?
네, 원하는 만큼 지원 가능합니다.

### GPS를 꼭 켜야 하나요?
필수는 아니지만, GPS를 켜면 가까운 일자리를 우선 추천받을 수 있습니다.

### 한국어 시험은 필수인가요?
필수는 아니지만, 시험 결과가 좋으면 매칭 점수가 높아집니다.

## 🏢 기업

### 외국인 채용이 처음인데 가능한가요?
네! 국가별 채용 가이드와 비자 정보를 제공합니다.

### 지원자의 한국어 실력을 어떻게 확인하나요?
- 한국어 시험 결과 확인
- 직접 시험 문제 출제 가능
- 채팅으로 직접 확인

### 플래티넘 패스란 무엇인가요?
프리미엄 서비스로 우선 노출, 무제한 메시지, 상세 분석 등을 제공합니다.

### 공고는 얼마나 오래 노출되나요?
기본 30일, 연장 가능합니다.

### 수수료가 있나요?
- 공고 등록: 무료
- 채용 성공 수수료: 없음
- 프리미엄 기능만 유료

## 🔒 안전 및 보안

### 개인정보는 안전한가요?
ISMS-P 인증을 받았으며, 모든 데이터는 암호화되어 저장됩니다.

### 가짜 공고나 스팸은 어떻게 관리하나요?
- 24시간 모니터링
- AI 스팸 필터링
- 신고 시스템 운영
- 기업 인증 절차

### 채팅 내용은 저장되나요?
네, 분쟁 해결을 위해 90일간 보관 후 자동 삭제됩니다.

## 💳 결제

### 어떤 결제 수단을 사용할 수 있나요?
- 신용카드/체크카드
- 계좌이체
- 앱스토어/구글플레이 결제

### 환불이 가능한가요?
유료 서비스 이용 전 7일 이내 전액 환불 가능합니다.

### 해외 카드로도 결제 가능한가요?
네, 가능합니다.

이용약관 위반 시 정지될 수 있습니다. support@linkjob.kr로 문의하세요.
