## 서비스 소개

### 한 줄 소개
> 외국인 구직자와 한국 기업을 GPS 기반 AI 매칭으로 연결하는 모바일 채용 플랫폼

### 엘리베이터 피치
LinkJob은 한국에서 일자리를 찾는 외국인과 외국인 직원을 채용하려는 기업을 연결하는 모바일 플랫폼입니다. GPS 기반 매칭, AI 추천 시스템, 12개 언어 지원, 한국어 능력 검증 시스템을 통해 언어 장벽을 넘어 최적의 인재를 연결합니다.

### 주요 특징
1. **GPS 기반 근거리 매칭** - 5km 이내 일자리 우선 추천
2. **AI 매칭 스코어** - 0-100점 정밀 적합도 계산
3. **12개 언어 지원** - 실시간 번역으로 소통 장벽 해소
4. **한국어 능력 검증** - 내장된 시험 시스템으로 실력 확인
