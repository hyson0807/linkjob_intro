## 서비스 소개

### 한 줄 소개
> 한국말 검증된 외국인 구인구직 플랫폼

### 엘리베이터 피치
LinkJob은 한국에서 일자리를 찾는 외국인과 외국인 직원을 채용하려는 기업을 연결하는 모바일 플랫폼입니다. GPS 기반 매칭, 12개 언어 지원, 한국어 능력 검증 시스템을 통해 언어 장벽을 넘어 최적의 인재를 연결합니다.

### 주요 특징
1. **GPS 기반 근거리 매칭** - 5km 이내 일자리 우선 추천
3. **12개 언어 지원** - 실시간 번역으로 소통 장벽 해소
4. **한국어 능력 검증** - 내장된 시험 시스템으로 실력 확인
